# mockou

- 2D chess pieces inspired by handmade wooden chess sets, featuring cuteness and simplicity
- 手作りの木製チェスセットからインスピレーションを得た、かわいらしさとシンプルさが特徴の2Dチェス駒

## ボードを選ばない見やすさ

<img width="1920" height="1080" alt="mockou_sample_3" src="https://github.com/user-attachments/assets/1c6a2a01-1dbd-4a5e-93d9-3791feb38048" />


- 白いコマだけでなく、黒いコマにもフチ取りがあります。
- もしボードの色が暗かったとしても、コマを識別しやすくなっています。すなわち、コマのためにボードの選択肢が減ることはありません。

## ポーンのバリエーション

<img width="1920" height="1080" alt="mockou_sample_2" src="https://github.com/user-attachments/assets/d5eeb847-4c90-4846-b06b-8a51bc2f5f6b" />


- ポーンのデザインとして、wP（白のポーン）・bP（黒のポーン）の他に、wP2・bP2を用意しています。
- READMEの最上部にあるサンプル画像では、wP2・bP2を使用しています。
- wP2・bP2は、他の駒に比べて下部の幅が狭くなっており、ポーン以外の駒との区別が付きやすくなっています。
- wP・bPは他の駒と下部の幅が同じで、こちらを使用すると視覚的に統一感が出ます。

## 名前の由来

<img width="1920" height="1080" alt="mockou_sample_4" src="https://github.com/user-attachments/assets/87836149-e58e-4c9d-bb3c-439118b5f82a" />

- Mockouの名前は、日本語の「木工」と英語の“mock”のかばん語です。これは、ナイトのデザインが、実在する木製チェスセットのまねごと（mock）だからです。
