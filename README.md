## Pawn variations ポーンのバリエーション

For pawn designs, we offer wP (white pawn) and bP (black pawn), in addition to wP2 and bP2. The sample image at the top of the README utilizes wP2 and bP2. The wP2 and bP2 designs feature a narrower base compared to other pieces, making it easier to distinguish them from non-pawn pieces. In contrast, wP and bP have the same base width as other pieces, which provides a visually uniform appearance when used.

ポーンのデザインとして、wP（白のポーン）・bP（黒のポーン）の他に、wP2・bP2を用意しています。READMEの最上部にあるサンプル画像では、wP2・bP2を使用しています。wP2・bP2は、他の駒に比べて下部の幅が狭くなっており、ポーン以外の駒との区別が付きやすくなっています。wP・bPは他の駒と下部の幅が同じで、こちらを使用すると視覚的に統一感が出ます。

##  名前の由来

The name Mockou is a portmanteau of the Japanese word “mokkou” (木工; woodworking) and the English word “mock”. This is because the knight's design is a _mock_ version of an actual wooden chess set.

Mockouの名前は、日本語の「木工」と英語の“mock”のかばん語です。これは、ナイトのデザインが、実在する木製チェスセットのまねごと（mock）だからです。
